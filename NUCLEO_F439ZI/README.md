# NUCLE-F439ZI

Devboard's web: https://os.mbed.com/platforms/ST-Nucleo-F439ZI/


## Wiring

Use TROPIC01 shield, or wire chip according to following diagram:

```
|------------|-----------|
|  TROPIC01  |   NUCLEO  |
|------------|-----------|
|  GND       |  GND      |
|  3V3       |  3V3      |
|  MISO      |  GPIOE_5  |
|  MOSI      |  GPIOE_6  |
|  SCK       |  GPIOE_2  |
|  CS        |  GPIOE_4  |
|------------|-----------|
```
