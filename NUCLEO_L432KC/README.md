# NUCLE-L432KC

Devboard's web: https://os.mbed.com/platforms/ST-Nucleo-L432KC/


## Wiring

Use TROPIC01 shield, or wire chip according to following diagram:

```
|------------|-----------|
|  TROPIC01  |   NUCLEO  |
|------------|-----------|
|  GND       |  GND      |
|  3V3       |  3V3      |
|  MISO      |  GPIOA_6  |
|  MOSI      |  GPIOA_7  |
|  SCK       |  GPIOA_5  |
|  CS        |  GPIOA_4  |
|------------|-----------|
```
